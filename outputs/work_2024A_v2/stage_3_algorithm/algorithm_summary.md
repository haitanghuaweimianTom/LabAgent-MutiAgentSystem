算法名称：等距螺线舞龙轨迹求解算法

输入数据：螺距p=0.55m、龙头板长L_head=3.41m、龙身板长L_body=2.20m、板宽d=0.275m、把手孔距l_head/l_body、龙头速度vh=1.0m/s、总节数N=224节、初始半径r0=16p、总角度Theta_total。

核心步骤：
1. 建立阿基米德螺线模型：定义极坐标方程r(θ)=r0-aθ（a=p/2π），推导直角坐标位置spiral_pos及导数spiral_deriv；
2. 弧长积分与反解：通过数值积分arc_length计算螺线弧长，采用Newton迭代法find_theta_for_arc由目标弧长反解对应极角θ；
3. 递推求解各节位置：基于弧长约束（龙头l_head、龙身l_body），逐节建立非线性方程组，用fsolve求解相邻节角度差，实现全龙身坐标追踪。

输出格式：JSON文件存储各时刻t下224节板凳的(x,y)坐标矩阵，维度为T×N×2，附带速度、碰撞检测等辅助字段。

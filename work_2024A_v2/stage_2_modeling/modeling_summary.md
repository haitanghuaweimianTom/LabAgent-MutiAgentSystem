等距螺线盘入运动学建模与数值求解——建模摘要

【核心变量】
- $p = 0.55$ m（螺距）：螺线相邻两圈径向间距，决定螺旋紧密程度
- $a = p/(2\pi) \approx 0.0875$ m/rad（螺线系数）：极径随转角变化率，表征螺线"松紧"
- $r_0 = 8.8$ m（初始半径）：龙头起始位置极径，对应第16圈
- $v_h = 1$ m/s（龙头前把手速度）：恒定驱动速度，系统运动学边界条件
- $l_j$（孔间距）：相邻把手点约束长度，$l_1 = 2.86$ m（龙头），$l_j = 1.65$ m（$j\geq 2$，龙身）

【核心公式】
等距螺线极坐标方程（避免负半径）：
$$r(\theta) = a(\Theta_{\text{total}} - \theta) = r_0 - \frac{p}{2\pi}\theta, \quad \theta \in [0, 32\pi]$$
弧长微分方程（龙头速度约束）：
$$\frac{ds}{dt} = v_h \Rightarrow \frac{d\theta}{dt} = -\frac{v_h}{\sqrt{r^2 + a^2}} = -\frac{v_h}{a\sqrt{(\Theta_{\text{total}}-\theta)^2 + 1}}$$

【求解策略】
以龙头把手为基准，通过弧长-转角关系数值积分确定龙头实时位置，再基于固定孔间距$l_j$的几何约束逐节递推求解各把手点坐标。

【与子问题的映射】
- 问题一（盘入轨迹与速度）：直接应用上述运动学模型，数值求解$\theta(t)$后递推全链位置，并通过速度合成公式$v_j = |\dot{\vec{P}}_j|$计算各把手速度；
- 问题二（碰撞判定与终止时刻）：在位置递推基础上引入板凳矩形几何与相邻节夹角约束$\phi_j$，通过距离检测算法确定板凳间首次碰撞的临界时刻；
- 问题三（最小螺距优化）：将$p$由固定参数转为优化变量，以碰撞临界半径为约束，通过一维搜索求解使盘入过程恰好不发生碰撞的最小$p$值；
- 问题四（调头曲线设计）：在螺线中心区域构造由两段相切圆弧组成的S形调头曲线，保证与盘入、盘出螺线光滑衔接，优化目标为调头曲线最短；
- 问题五（龙头最大速度约束）：以问题四调头路径为几何约束，建立全链速度传递模型$v_j = f(v_h, \kappa_j)$，通过曲率$\kappa_j$极值点确定龙头速度$v_h$的上界使所有$v_j \leq 2$ m/s。

**算法名称**：SiC薄膜反射光谱双光束干涉反演算法

**输入数据**：Excel光谱文件（含波长列与反射率列）、SiC掺杂浓度参数（默认1e16 cm⁻³）

**核心步骤**：
1. **数据解析**：自动识别波长/反射率列名（支持中英文关键词匹配），提取实验光谱数据；
2. **色散计算**：基于Sellmeier方程计算SiC复折射率n(λ)，考虑掺杂浓度修正；
3. **干涉建模**：构建空气-SiC-衬底双光束干涉模型，计算正入射/斜入射条件下的界面菲涅尔反射系数与光程差，拟合理论反射光谱。

**输出格式**：理论反射率数组R(λ)，用于与实验数据最小二乘优化拟合，反演薄膜厚度d及光学常数。
